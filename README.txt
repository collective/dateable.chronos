dateable.chronos
================

Dateable.chronos is a set of calendar views for Plone.

Features
--------

Day, Week and Month calendar views. Also an upcoming list and past list.


Installation
------------

Installation is done via the installer of the bundling product, i.e. 
p4a.plonecalendar.


Requirements
------------

* dateable.kalends

* Plone.

* It's currently developed as a part of p4a.plonecalendar, but is in theory
  usable with other integrations with Plone.
